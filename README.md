### Setup
```bash
$ git clone https://github.com/mkmayank/arusank.com
$ cd arusank.com
$ git submodule init
```

### Adding content
> add content in hugo/content directory

### Build
```bash
$ cd hugo
# for letting hugo build files automatically
$ hugo server -D
# to build pages
$ hugo -D
```

### Publish
* build pages
* commit changes and push
